Example

WWW.ANIVIAGODS.NET

Anivia is a champion in the game League of Legends. Being one of the best Anivia midlaners in the world, I took it upon myself to create a fully automated site for the Anivia community which uses the League of Legends API and node scripts set on a timer to retrieve the builds and playstyles of the most successful players of Anivia accross North America, Europe, and Korea. The site is an easily digestable way to see what items and runes the most successful and highest ranking players are using so you can try them yourself.
